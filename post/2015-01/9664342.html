유튜브 동영상 720P,1080P 끊김없이 보는 법


※ 크롬 사용자 분들은 익스로도 재생해보시고 문제 있을 시 설정해 주시면 될 것 같아요.
   크롬의 플래시 작동은 익스보다 불안정한지 제 컴에서는 1024p 까지만 잘 작동되고 그 이상 가면 끊김이 심하네요.
   결론 : 유튜브 고화질은 익스로 감상하시는 게...

UCC천국이라 할수있는 유투브 하지만! 유투브는 해외에서 운영되는 서버이기 때문에...

306P이상의 화질로 시청할경우 버퍼링이 아주 심각합니다. 열을 받은 나머지. 뮤비/영상을 시청하는 것은

포기 하려던중 DNS를 이용하면, 버퍼링 없이 720p, 1080p 급의 동영상도 시청할 수 있다는것을 알았습니다.

자 그럼 지금부터 DNS를 설정하는 방법에 대해서 알아보겠습니다.
참고로 Windows7 OS를 기준으로 작성 하겠습니다.

* 제어판에 들어갑니다. 제어판은, 시작메뉴 → 제어판을 통해 들어가실 수 있습니다


* 여기서네트워크 상태 및 작업 보기를 클릭해주시면 됩니다.

* "무선네트워크" 또는 "로컬 영역 연결"을 클릭하시면 됩니다.



* 이미지 좌: 연결 상태에 들어가셨으면, 속성(P) 버튼을 눌러주세요.

이미지 우: Internet Protocol Version 4 (TCP/IPv4) 를 활성화 시키신뒤. 속성(R) 버튼을 눌러주세요.


* 기본 설정 DNS 서버에 8.8.8.8 을
보조 DNS 서버에 8.8.4.4 를 입력한뒤 확인 버튼을 눌러주세요.

* 마지막으로 시작 메뉴 -> 검색에서 cmd 를 입력 하시거나. 윈도우키+ R을입력하셔서,

도스창을 실행 시켜주시기 바랍니다.

도스창에 ipconfig /flushdns 를 입력하시고. 엔터!


이제 dns 초기화까지 완료 되었습니다.

인터넷을 재실행 하시고 유투브나, 구글 관련 사이트에 접속 하면! 끊김없이 브라우징이 되는 것을 아실 수 있습니다.

[펌] (SSD최적화)OS재설치 없이 AHCI모드로 변경(WIN7)

요즘 SSD최적화에 대해 공부중인데 
그중 메인보드 메뉴에서 SATA모드를 IDE에서 AHCI로 바꾸니 블루스크린이 떠버렸습니다;
그래서 포맷없이 변경시키는방법을 기록해두려고 합니다.

*해당방법은 WIN7에서만 정상작동 확인했습니다.
*AHCI모드로 사용하기 위해서는 메인보드에서 AHCI를 지원해야 가능합니다. 지원여부는 사용중인 메인보드의 매뉴얼 등 스펙을 확인해 주시고 미지원인 경우에도 가능한 경우가 있으니 참고 하시기 바라겠습니다.(제경우는 인텔칩셋 메인보드입니다.)

이 방법은 비스타와 윈7에만 해당되는 방법으로 OS에 AHCI 드라이버가 내장되어져 있어 가능한 방법입니다.
 
방법은 다음과 같습니다.(기존 IDE모드에서 진행)
 
1. 시작 => 실행창에 "regedit"를 입력 실행시켜 줍니다.
 
2. 사용권한 메세지가 나타나는 경우 "계속"을 선택해 줍니다.
 
3. 레지스트리편집기가 실행되면"HKEY_LOCAL_MACHINE/SYSTEM/CurrentControlset/services/msahci"으로이동합니다.

4. "start"를 더블클릭해서 팝업되는 창에서 값 데이터를 "0"으로 변경합니다.
 
재부팅을 하면서 CMOS상에서 IDE모드를 AHCI모드로 변경합니다.

OS진입이 잘되는지 확인합니다.
(OS진입후 재부팅이 1-2회 정도 있을 수 있습니다.)
